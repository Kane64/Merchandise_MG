## レトルトカレー専門店の商品管理システム

### 概要

このシステムでは店舗で扱う商品の在庫管理を行うことが出来ます。
商品の新規登録から編集、削除を行うことができ、各商品の在庫を調整することが出来ます。

店内でお客様からリクエストがあった商品を検索するというシチュエーションを想定し、新入スタッフの方にも商品が一目瞭然になるように画像をメインに配置しました。

### 主な機能
- ログイン・ログアウト機能
- 商品一覧画面
- 商品新規登録、編集、削除機能
- 商品在庫管理機能
- 商品検索機能
- ソート機能
- ページネーション機能
- 画像ファイルアップロード機能

## 開発環境
